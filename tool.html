What we use to create this projects?

Explore now!!!!

Thanks to Github, we can have a workspace for coding and sharing it with other members. GitHub allows everyone to keep track of what other members are working on, and because the history commit is being recorded all the time, we can easily know who is the last one to fix the code and what they edit. In some cases, if anyone accidentally deletes some code, Github makes finding and fixing it way easier. GitHub is also a transport for us to the public prototype so that everyone can take a look by themselves.
